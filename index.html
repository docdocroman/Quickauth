An authentification without sign in/up


This is a authentification concept initially created for a mobile application ( Five Seconds, not available yet).

This concept allows to keep user logic in an application (any kind) but remove the "sign up", "sign in" system for the user. More friendly for the user, simple to integrate in code and the same "user logic" in your app.
